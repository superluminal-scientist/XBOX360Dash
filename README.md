An HTML+CSS+JS version of my own personal version of https://github.com/ZivvoZ/dashx360. I've been re-doing the entire front-end in web stuff so that it's easier to work with and expand for my own use in my sleeper 360.

It's genuinely nothing special. Just a faithful port. On the actual software end (the one that isn't this repo), I have a version of the dashx360 C# backend that talks back and forth with this, and renders via a WebView2. Neat, right?
